rails-cache_control
===================

Automatically set Cache-control header when [caches_action expires_in](https://github.com/rails/actionpack-action_caching) specified.

Requirements
============

Ruby >= 2.0.0
Rails >= 4.0.0

Installation
============

Include the gem in your Gemfile:

```ruby
gem 'rails-cache_control'
